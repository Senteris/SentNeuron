# Regression analysis
Neural Network for regression data
## Usage
### train.py
Put data to `./data`  
For example:  
```
regression
└───data
    └───Timestamp
    └───Open
    └───High
    └───Low
    └───Close
    └───Volume_(BTC)
    └───Volume_(Currency)
    └───Weighted_Price
```
Some datasets:
- https://www.kaggle.com/mczielinski/bitcoin-historical-data?select=bitstampUSD_1-min_data_2012-01-01_to_2020-04-22.csv

And start train model
